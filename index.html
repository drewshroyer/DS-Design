TIPS
  • click any object from the furniture sidebar section to add it to the canvas
  • select an object to resize/rotate by clicking directly onto the object
  • deselect by clicking on a new object or any blank spot in the canvas
  • objects may be dragged with or without being heighlighted
  • delete objects by dragging them over to the furniture selection gallery
  • use the 'CLEAR' navigation button to clear all objects from the canvas
  • use the 'NEW' navigation button to redefine the shape of your room
  • if you close these tips, but get lost, easily rerender using the "TIPS" navigation button
